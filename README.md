# Chetacare

Chetacare is a healthcare-focused frontend website built to present a clearer, more supportive care experience for people managing chronic health needs. The site introduces the brand, explains how the service works, shares health content through a blog, and gives users a simple path to reach the team.

## What This Project Includes

- A marketing-style home page with sections for the product story, target audience, current focus, testimonials, and blog highlights
- Supporting pages for About, Contact, FAQ, Privacy, and Terms
- A blog listing page and individual blog post pages with related article suggestions
- Responsive layouts built for mobile and desktop browsing

## A Few Things About Chetacare

- The messaging is centered on preventive care and helping people get support before issues become emergencies
- The experience is designed to feel trustworthy, simple, and approachable
- The blog is part of the product story, giving users practical health information alongside the main brand pages

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Lucide React

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Preview the production build:

```bash
npm run preview
```

## Project Structure

```text
src/
  components/   Reusable UI sections and layout pieces
  data/         Static blog content and supporting data
  pages/        Route-level pages
```

## Notes

This repository currently focuses on the frontend experience. Most content is statically defined in the app, including the blog posts and page copy.
